# Changelog Page for Marketing Site

## Context

LaxDB needs a changelog page to communicate platform updates — new league data, features, improvements. Inspired by [linear.app/changelog](https://linear.app/changelog) but adapted to LaxDB's existing design language (dashed SVG lines, serif italic headings, orange accents, staggered fade animations).

## Data Model (`lib/changelog.ts`)

```ts
type ChangelogCategory = "data" | "feature" | "improvement" | "community"

interface ChangelogEntry {
  id: string
  title: string
  date: string              // ISO YYYY-MM-DD
  category: ChangelogCategory
  summary: string           // 1-2 sentences below title
  body: string[]            // Paragraphs
  highlights?: string[]     // Optional bullet points
}
```

Static array of entries. No content-collections / MDX — overkill for a changelog v1.

Filter categories: All | Data | Features | Improvements | Community (same tab pattern as `blog/index.tsx` lines 7-12, 52-74).

## Layout Design

### Page Header
- `max-w-4xl mx-auto px-4 py-16 md:py-32` (wider than blog's `max-w-screen-sm`, narrower than homepage's `max-w-6xl`)
- Title: `h1.font-serif.text-2xl.italic` (matches blog index and about page h1 patterns)
- Filter tabs: Exact same `border-b border-border` + active underline pattern from blog index

### Timeline (the core design element)

Desktop (md+) — asymmetric two-column:

```
  Jan 15, 2026    ●─── PLL 2025 Season Data
                  ┆    DATA
                  ┆
                  ┆    Complete game-by-game statistics...
                  ┆
                  ┆    • 156 games across 8 teams
                  ┆    • Player stats: G, A, P, GB, CT
                  ┆
  Dec 2, 2025     ●─── Knowledge Graph Explorer
                  ┆    FEATURE
```

- **Timeline spine**: Dashed SVG vertical line reused from `features.tsx` lines 29-47 (`<line strokeDasharray="3 3" strokeWidth="2" className="stroke-border">` with gradient mask at top/bottom)
- **Orange dot markers**: `size-2.5 rounded-full bg-orange ring-4 ring-background` — punches through the dashed line
- **Date column**: `w-[140px] shrink-0 text-right` — `text-xs text-subtle whitespace-nowrap` using `formatPublishedDate` from `@/lib/date`
- **Content column**: `flex-1 min-w-0 pb-16`

Mobile (<md): Single column. Timeline line at left edge, date above content.

### Entry Design

- **Title**: `font-serif text-xl md:text-2xl text-foreground italic` (LaxDB's serif signature)
- **Category**: `.text-label text-xs` (uppercase, spaced — existing utility in globals.css)
- **Summary**: `mt-3 text-[15px] leading-relaxed text-muted-foreground`
- **Body**: `mt-4 space-y-3 text-[15px] leading-relaxed text-foreground` (matches about.tsx)
- **Highlights**: Bullet list using about.tsx's pattern — `<span className="mt-1.5 block size-1.5 shrink-0 rounded-full bg-bullet" />` (line 52)

### How It Differs from Linear

| Linear | LaxDB |
|--------|-------|
| Solid timeline line | Dashed SVG line (site signature) |
| Sans-serif titles | Serif italic (Newsreader) |
| Multi-colored category dots | Orange dots + uppercase text labels |
| Dark-only | Light + dark via CSS variables |
| Collapsible sections | All content visible |
| "Now" title | "Changelog" |

## Animation

- Entire page in `FadeContainer` (staggered children: 50ms gaps, 200ms delay)
- Title, filter tabs, and each entry wrapped in `FadeDiv`
- Spring physics: stiffness 150, damping 19, mass 1.2 (existing config)
- No scroll-triggered animations — entrance only
